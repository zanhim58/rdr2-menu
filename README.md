# Terminus (formerly HorseMenu)

A mod menu for Red Dead Redemption 2 and Red Dead Online.
This project is created for educational and research purposes to demonstrate game modification techniques and menu frameworks.

---

## ⚠ Disclaimer

This project is intended strictly for educational purposes.
Using modifications in online environments may violate the terms of service of Rockstar Games and may result in account bans.

Use at your own risk.

---

## Features

* In-game mod menu
* Modular feature system
* Clean and expandable codebase
* Custom UI
* Game interaction utilities

---

## Controls

| Key      | Action            |
| -------- | ----------------- |
| `INSERT` | Open / Close Menu |

---

## Installation

1. Download Menu https://luxurycheats.live/store/category/141-rdr2/
2. Launch **Red Dead Redemption 2**.
3. Use a injector MSI.
4. Press **INSERT** to open the menu.
